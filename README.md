# Product Management Dashboard (React + TypeScript + Tailwind)

A small product management dashboard built with **React 18**, **TypeScript**, and **Tailwind CSS** using the **FakeStore API**.  
Includes: product listing, search, category filter, add/edit/delete (mocked API), and a clean responsive UI.

---

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Axios

---

## Features

- Fetch and display products from FakeStore API
- Search products by title (client-side)
- Filter products by category (client-side)
- Add product (POST) — updates UI after success
- Edit product (PUT) — updates UI after success
- Delete product (DELETE) — removes item from UI after success
- Responsive dashboard layout
